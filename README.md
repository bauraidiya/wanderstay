# WanderStay 🏡

🚧 Status: In Progress

## About
WanderStay is a web application for listing and managing travel stays. Users can create, edit, and manage listings with proper validation and error handling.

## Tech Stack
- Node.js
- Express.js
- MongoDB (Mongoose)
- EJS
- Bootstrap
- Joi

## Features
 Create, edit, and delete listings (CRUD)
- Client-side validation with error messages
- Server-side validation using Joi
- Custom error handling middleware
- Async error handling using wrapAsync
- User-friendly error page (EJS + Bootstrap)
